# Playbook: design

The site itself is evidence. A founder judges the craft of the page as a
proxy for the craft of the person. Every screen must pass the "who built
this?" bar and still pass the brand wow test in ../positioning.md.

Everything else about visual work lives with the Designer:
[../../designer/README.md](../../designer/README.md). Read its routing
table for direction, motion, the design language, the design system
registry, and the mandatory screenshot critique. The engineering rules
(100-line files, single responsibility, reuse primitives, Biome and
TypeScript clean) stay in the root CLAUDE.md.
